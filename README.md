### Python Script to Explore US Bikeshare Data
This project uses Python to explore and understand data related to bikeshare systems in Chicago, New York City, and Washington.

### How to Run this Script
You can run the script using a Python integrated development environment (IDE). This script is written in Python 3, so you will need the Python 3.x version of the installer.

### Datasets
Data for the first six months of 2017 are provided for all three cities.  The data files contain six columns from:
* Start Time
* End Time
* Trip Duration
* Start Station
* End Station
* USer Type

The Chicago and New York City files contain the following additional columns:
* Gender
* Birth

### Files
In order to run the program, the following files are needed:
* chicago.csv
* new_york_city.csv
* washington.csv

### Statistics Computed

Statistics on most popular times of travel
* Most common start hour
* Most common day of week
* Most common month

Statistics on most popular stations and trip
* Most commonly used start station
* Most commonly used end station
* Top 5 most common start/stop station trips

Statistics on trip durations
* Total number of trips
* Total travel time
* Average travel time

Statistics on users
* Counts of user types
* Counts of genders
* Birth years (earliest, most recent, and most common)
